# Start Budgeter.
```
git clone https://github.com/osama0001/budgeter.git
```
# get inside the project

# install the required packages
```
npm install
```
# start the app on http://localhost:8080
```
npm run serve
```
# get into node server
```
cd src/server
```
# add required packages
```
npm install
```
# start database
```
mongo
```
# Start node server
```
nodemon
```
